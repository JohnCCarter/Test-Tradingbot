A Python-based trading bot template.

## Installation

Clone the repository and install dependencies:

```bash
conda env create -f environment.yml
```

## Usage

Run the bot:

```bash
python src/tradingbot.py
```

## Testing

Run tests:

```bash
pytest tests/
```
